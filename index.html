How to contribute

Use the contributing guidelines in each semester repo. To avoid downloading everything, use git sparse-checkout or access single files from the GitHub UI.

  • Fork the semester repo you want to update.
  • Use a branch like add-week-3-notes.
  • Open a PR with a clear description and small file sizes.
